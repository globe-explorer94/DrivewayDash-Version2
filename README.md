# DrivewayDash

Fast winter help for the GTA commuter. Shoveling, Salting, and Scaping on-demand.

## Market Focus
Western/Northern GTA & Guelph:
- Mississauga
- Oakville
- Brampton
- Markham
- Vaughan
- Richmond Hill
- Burlington
- Milton
- Guelph

## Setup
1. Install dependencies: `npm install`
2. Run development server: `npm run dev`

## Deployment
The app is designed to be a single-file React application for easy deployment.
A `netlify.toml` is included for Netlify routing support.
